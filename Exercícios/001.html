Olá, Mundo!

A MALDIÇÃO DO OLÁ MUNDO

A lenda do Olá Mundo (em inglês "Hello, World!") diz que caso o primeiro código do programador não seja essa mensagem, o mesmo não aprenderá ou não conseguirá usar tal linguagem de programação.

Este código foi escrito em 1972 por Brian Kernighan em seu livro The C Programming Language. A partir de então, o termo viralizou entre os programadores, tanto que, atualmente, é muito popular entre a comunidade de desenvolvedores e até por instituições profissionais de ensino.
